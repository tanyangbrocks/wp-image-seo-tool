WordPress 疊字圖片產生器

Created by 彈羊如柳丁

上傳圖片後會自動 OCR 辨識圖片裡的文字，連同文字在圖片上的位置、大小、實際顏色一起還原成疊在圖片上的真 HTML 文字（可被選取），顏色是分析原圖文字像素抓出來的真實色碼，不是只判斷黑白。
辨識完成後右邊會立刻出現可以直接調整的編輯畫布（拖曳移動、選取後縮放、點文字直接打字修改），調好之後填 Alt 描述、按「產生 HTML」即可。
需要連網（第一次使用會下載辨識模型，之後同語言不用再下載）。

金鑰只會從你的瀏覽器直接打到 Google 的伺服器，不會經過任何第三方、也不會存進產生的 HTML 裡。 每個人要用這個功能都要貼自己的金鑰，不要把你的金鑰分享給別人。 辨識失敗時（金鑰錯誤/網路問題）會自動改用 PaddleOCR。
