WordPress 疊字圖片產生器

上傳圖片後會自動 OCR 辨識圖片裡的文字，連同文字在圖片上的位置、大小、實際顏色一起還原成疊在圖片上的真 HTML 文字（可被選取），顏色是分析原圖文字像素抓出來的真實色碼，不是只判斷黑白。
自動處理好之後，填 Alt 描述再按「產生 HTML」，就會產生完整 HTML，同時顯示 WordPress 會怎麼渲染它的預覽。
需要連網（第一次使用會下載辨識模型，之後同語言不用再下載）。

語言設定：勾選圖片裡包含的語言（影響辨識依據，勾越少通常辨識越快越準）
進階設定：使用自己的 API 金鑰提升辨識準確度（選填）

金鑰只會從你的瀏覽器直接打到 Google 的伺服器，不會經過任何第三方、也不會存進產生的 HTML 裡。 每個人要用這個功能都要貼自己的金鑰，不要把你的金鑰分享給別人。 沒有金鑰或沒勾選的話，會照常使用免費的離線辨識，不影響原本的功能。
